# 🍺 Flunkyball Tournament Tracker

A single-file, offline-capable tournament tracker for Flunkyball. No backend, no login, no internet required after first load.

---

## Hosting on GitHub Pages

### 1. Create a new repository

1. Go to [github.com](https://github.com) and sign in.
2. Click **New repository**.
3. Name it `flunkyball-tracker`.
4. Leave it **Public**.
5. Click **Create repository**.

### 2. Upload the files

1. Click **Add file → Upload files**.
2. Upload `index.html` and `README.md`.
3. Click **Commit changes**.

### 3. Enable GitHub Pages

1. Go to **Settings → Pages**.
2. Under **Build and deployment**, select **Deploy from a branch**.
3. Set branch to **main** and folder to **/ (root)**.
4. Click **Save**.
5. After a minute, GitHub shows your site URL (e.g. `https://yourusername.github.io/flunkyball-tracker/`).

---

## Opening on iPhone

1. Open Safari on your iPhone.
2. Navigate to your GitHub Pages URL.
3. Tap the **Share** button (rectangle with arrow).
4. Tap **Add to Home Screen**.
5. Tap **Add**.

The app now opens like a native app — full screen, no browser chrome.

---

## Important: Where your data lives

> ⚠️ Tournament data is stored **locally on your phone** in Safari's localStorage. It is **not** stored on GitHub or any server.

- **If you clear Safari data**, your tournament history will be lost.
- **If you switch to a different device or browser**, you start fresh.
- Use **💾 Backup → Export JSON** regularly to save a copy of your data.

---

## Backup tips

- Export a JSON backup after each day of play.
- AirDrop or email the JSON file to yourself for safekeeping.
- To restore: paste the JSON into **Backup → Import Backup**.

---

## Troubleshooting

| Problem | Solution |
|---|---|
| Page shows 404 | Wait 1–2 minutes after enabling Pages, then refresh |
| Data disappeared | Restore via **Backup → Import Backup** using your JSON export |
| App looks broken after update | Hard-refresh: hold Reload in Safari |
| GitHub hosts the app, not your stats | Your match data lives only on your device |
